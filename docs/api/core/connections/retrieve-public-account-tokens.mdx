# List connection public account tokens

**About**: Returns all the public account tokens associated with a connection or an empty array.

**API Method**: `GET /connections/:connectionId/public_account_tokens`

#### Example
<ConnectionsRetrievePublicAccountTokens />

### Response

{

  ]
}
