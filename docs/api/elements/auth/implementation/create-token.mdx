# Create element token

**About**: Creates an element token to launch an Auth Element session for a specific
entity. Element tokens are only valid for 30 minutes after they are created, and can
only be used once.

**API Method**: `POST /elements/token`

### Request

<ElementsCreateAuthToken />

#### Parameters
| Name          | Type     | Required | Description                                                                      |
|---------------|----------|----------|----------------------------------------------------------------------------------|
| `entity_id`   | `string` | ✅        | The id of the end user (`entity`) for which the `element_token` will be created. |
| `team_name`   | `string` | ✅        | The name used to refer to your company/app throughout the auth flow.             |
| `type`        | `string` | ✅        | The type of element you wish to launch. In this case `auth`.                     |
| `auth`        | `object` | ✅        | Auth element information.                                                        |

### Response

#### Example

```json
{
  "element_token": "pk_elem_f3TZsKnmP7Q69bX65hRHngBEb8mT8gbG"
}
```
