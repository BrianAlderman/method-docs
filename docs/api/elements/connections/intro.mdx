# Overview of the Connections Element

The Connections Element allows your users to securely connect and manage their Method accounts through your app. This implements the same tokenization, and account creation used by the Link Element.

### Connections Element Flow Overview
The Connections Element flow begins once your user is required to manage their connected accounts. This may include adding more accounts through the link element, or simply viewing their connected accounts.
